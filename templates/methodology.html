Random Forest is a popular machine learning algorithm that belongs to the supervised learning technique.       
Random Forest is an ensemble technique capable of performing both regression and classification tasks with the use of multiple decision trees and a technique called Bootstrap and Aggregation, also known as bagging.
The basic idea behind this is to combine multiple decision trees in determining the final output rather than relying on individual decision trees. Random Forest has multiple decision trees as base learning models. Random performance of row sampling and feature sampling from the dataset forming sample datasets for every model. This part is called Bootstrap.
